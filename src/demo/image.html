Image Selection with Preview

Select images and display them with live preview

About This Demo

This demo shows how to handle image file selection with validation and preview. When you select a file, the demo checks if it's an image and displays a preview.

Code Example

import ElFinderPicker from 'elfinder-picker';

const picker = new ElFinderPicker({
  url: '/path/to/elfinder'
});

picker.open((file) => {
  // Validate it's an image
  if (file.mime && file.mime.startsWith('image/')) {
    // Display image preview
    const img = document.createElement('img');
    img.src = file.url;
    document.body.appendChild(img);
  } else {
    alert('Please select an image file');
  }
}, null, { type: 'image' });
