Reusable Picker Instance

Create once, use multiple times for better performance

About This Demo

This demo shows how to create a single picker instance and reuse it for multiple file selections. This approach is more efficient than creating a new instance each time.

Try It Out

Click the button multiple times to select different files. Notice that the selection count increments.

Selection count: 0

Code Example

import ElFinderPicker from 'elfinder-picker';

// Create picker instance ONCE
const picker = new ElFinderPicker({
  url: '/path/to/elfinder'
});

// Reuse the same instance multiple times
button.addEventListener('click', () => {
  picker.open((file) => {
    console.log('Selected:', file);
    // Handle file selection
  });
});

// The picker DOM is created once and shown/hidden
// This is more efficient than creating new instances
