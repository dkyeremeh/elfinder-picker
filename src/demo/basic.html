Basic File Selection

Simple file picker with minimal configuration

About This Demo

This demonstrates the simplest use case of elFinder Picker. Click the button below to open the file picker and select any file. The selected file information will be displayed below.

Code Example

import ElFinderPicker from 'elfinder-picker';

// Create picker instance
const picker = new ElFinderPicker({
  url: '/path/to/elfinder'
});

// Open picker with callback
picker.open((file) => {
  console.log('Selected:', file);
  // file.url - File URL
  // file.name - File name
  // file.mime - MIME type
});
